## ✏️ Room Renaming Guide

When renaming rooms, update the following fields:

1. **`name`** - Display name (can be in Thai or English)
2. **`description`** - Room description (should match the new name)
3. **`icon`** - Emoji that represents the room
4. **`id`** - ⚠️ **CAUTION:** Changing this will break existing URLs and references
5. **`tags`** - Update if the new name changes the room's focus
6. **`rules`** - Update if the new name requires different rules

### ⚠️ Important Notes

- **Room ID changes** will require:
  - Migration of existing posts/threads
  - Update of all hardcoded references
  - Redirect setup for old URLs
  
- **Name/Description changes** are safe and only affect display

- **Tag changes** may affect existing posts using old tags
